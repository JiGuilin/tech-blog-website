## 🐛 问题汇总

### 1. 博客列表无法加载
**原因**: 清理内联 JS 时误删了 `loadBlogPosts()` 函数  
**修复**: 创建 `js/blog.js` 包含完整的博客加载逻辑

### 2. 主题切换失效
**原因**: 主题切换函数被删除  
**修复**: 在 `js/blog.js` 中恢复主题切换功能

### 3. 统计 ID 不匹配
**原因**: HTML 使用 `total-github`, `total-hn`，JS 使用 `total-words`, `total-tags`  
**修复**: 更新 JS 使用正确的 ID

### 4. 缺少进度条元素
**原因**: HTML 中没有进度条 div  
**修复**: 在 body 开始处添加进度条元素
