## 🐛 根本原因

**博客列表页 (blog/index.html) 中有 428 行内联 `<style>` 块！**

这个内联样式块定义了固定的 `:root` CSS 变量：
```css
:root {
    --bg-primary: #0d1117;  /* 固定为暗色 */
    --text-primary: #f0f6fc;
    ...
}
```

这导致：
- ❌ 外部 CSS 的 `[data-theme="light"]` 变量被覆盖
- ❌ 主题切换时，内联样式优先级更高
- ❌ 只有按钮文字变化，实际颜色不变

---

## ✅ 修复方案

### 删除内联样式块
```bash
# 删除第 16-444 行的内联<style>
sed -i '16,444d' blog/index.html
```

### 使用外部 CSS
```html
<!-- 现在只引用外部 CSS -->
<link rel="stylesheet" href="../css/main.css">
```

## 🎯 技术说明

### CSS 优先级问题
```css
/* 内联样式（优先级高） */
<style>
  :root { --bg-primary: #0d1117; }
</style>

/* 外部 CSS（优先级低） */
[data-theme="light"] {
  --bg-primary: #ffffff;
}
```

**解决方案**: 删除内联样式，让外部 CSS 完全控制主题变量

### 主题切换流程
```javascript
1. 点击按钮 → toggleTheme()
2. 切换 data-theme 属性
3. CSS 变量自动应用新值
4. 页面颜色立即变化
```
